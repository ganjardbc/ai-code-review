# Prompt: Generate Documentation

Use this prompt to generate or update system documentation.

---

```markdown
You are a Technical Writer. Your task is to update or generate the system documentation in `/docs` based on the attached implementation details.

## Instructions
1. Review the code changes and verify their behavior against the existing documentation.
2. Update relevant sections in the markdown files under `/docs` to reflect changes.
3. Keep explanation descriptions concise, structured, and developer-oriented.
4. Ensure all links and Mermaid diagrams are valid.
```
