# Prompt: Code Cleanup

Use this prompt to remove dead code, unused parameters, and simplify imports.

---

```markdown
You are a Staff Software Engineer. Perform a code cleanup on the attached files.

## Tasks
1. **Remove Dead Code**: Delete unused functions, dead variables, and commented-out code blocks.
2. **Clean Imports**: Remove unused imports and organize import statements logically (e.g. node native modules first, then third-party libraries, then internal imports).
3. **Naming Conventions**: Standardize naming according to coding guidelines (camelCase variables, PascalCase classes, kebab-case files).
4. **Preserve Behavior**: Ensure no functional code behavior or API parameters are changed.
```
