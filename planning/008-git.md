# Goal
Implement the Git Operations adapter to manage cloning, checkout, and diff generation.

# Scope
Create the GitService adapter utilizing `execa` to clone repositories inside sandboxed directories, check out targets, generate diffs, and clean up.

# Prerequisites
* Node setup (`000-foundation`).
* Config loader (`001-configuration`).

# Deliverables
* `src/domain/interfaces/git.interface.ts` port interface.
* `src/infrastructure/git/git.service.ts` adapter implementation.

# Tasks
- [ ] Install packages: `pnpm add execa`
- [ ] Create `IGitService` interface in `src/domain/interfaces/git.interface.ts` defining methods: `clone()`, `checkout()`, `generateDiff()`.
- [ ] Implement `GitService` in `src/infrastructure/git/git.service.ts` using `execa`.
- [ ] Enforce **argument arrays** for all git command calls instead of constructing shell strings to prevent command injection (e.g. `execa('git', ['clone', '--depth=1', ...])`).
- [ ] Set up the cloning workflow to execute shallow, single-branch clones:
  ```typescript
  execa('git', ['clone', '--depth=1', '--single-branch', '--branch', sourceBranch, repoUrl, targetDir]);
  ```
- [ ] Implement `generateDiff` comparing target base and source branches. Ensure output is returned as standard string.
- [ ] Implement path checks to ensure paths are contained within `<WORKSPACE_DIR>` and prevent path traversal escapes.

# Acceptance Criteria
* Repositories clone successfully to local sandboxed folders.
* Command arguments are parsed safely (command injection attacks are blocked).
* Workspace path verification prevents path escapes.

# Testing Checklist
* **Unit Test**: Mock `execa` calls and verify command argument arrays.
* **Integration Test**: Run Git operations against a local test git repository and assert diff output.
* **Manual Test**: None.
* **Failure Scenarios**: Verify that cloning throws error if repository does not exist or branch is invalid.

# Risks
* Out-of-disk space from concurrent jobs. Ensure cleanup actions execute.

# Notes
Ensure git binaries are available on the worker environment path.
